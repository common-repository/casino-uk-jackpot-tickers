== Description ==

The Casino UK Jackpot Tickers plugin makes integrating progressive jackpots tickers into your casino affiliate site easy. Simply select the jackpots you want to display on your site, enter your affiliate code and you’re set!

**Currency, Font & Colour Options**

Choose from 50+ currencies, and select the font family, colour and size that best matches your site. You can also edit the custom CSS section to fine-tune the appearance.

**Logo Display Options**

Choose between an attractive game logo or just the progressive ticker.

**Widget & Shortcode Options**

Easily insert jackpot tickers in your sidebar using a widget or use the shortcode option to embed tickers in the copy of your site.

**Target Landing Pages**

Land your visitors on Casino UK’s [homepage](http://www.casinouk.com) or [progressive jackpots page](http://www.casinouk.com/progressive-jackpots.aspx).

**Important Note**

We are using external javascript file to retrieve the progressive jackpot tickers.

== Installation ==

1. Upload `casino-uk-jackpot-tickers.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Casino UK Jackpot Tickers menu and configure the plugin.
